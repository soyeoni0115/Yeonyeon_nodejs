# 9주차 

## API
- 다른 애플리케이션에서 현재 프로그램의 기능을 사용할 수 있게 접점을 의미

- 웹 API는 다른 웹 서비스의 기능을 사용하거나 자원을 가져올 수 있는 창구
- API를 '열었다', '닫았다'는 것 = 현재 기능을 허용했음

## 크롤링
- 웹사이트가 자체적으로 제공하는 API가 없거나 API 이용에 제한이 있을 때 사용하는 방법
- 표면적으로 웹 사이트의 정보를 일정 주기로 수집해 자체적으로 가공하는 기술


## 도메인
- 도메인 모델에는 인터넷 주소와 도메인 종류, 클라이언트 비밀키가 들어감
- 사용자 모델과 일대다 관계 갖음
